# 前端优化清单

## 骨架屏（Skeleton / Shimmer）

| 页面 | 触发时机 | 效果 |
|------|---------|------|
| Index.vue | 选择子标签后加载灵感列表 | 3 张闪烁骨架卡片 |
| InspireDetail.vue | 打开灵感详情 | 封面图 + 标题 + 正文骨架 |
| Search.vue | 热门灵感加载 | 3 条骨架列表项 |
| Personal.vue | 页面加载 | 3 张骨架卡片 |

动画：`shimmer` keyframe，灰白渐变来回移动

## 页面过渡动画

- 路由切换：淡入淡出 + 上下 10px 偏移
- 卡片出现：`fadeUp` 动画（自下而上 + 淡入）
- 骨架屏：闪烁渐变

## 回到顶部

- 右下角悬浮按钮，半透明白底 + 灰色箭头
- z-index: 999，不遮挡内容
- 滚动超过 400px 出现，点击平滑回到顶部
- hover 时加深阴影

## 无限滚动加载

- Index.vue 滚到底部自动加载下一页（每次 10 条）
- `hasMore` 标志控制停止，API 返回少于请求数时自动停止
- 加载中显示「加载更多...」，无数据显示「-- 没有更多了 --」

## 错误状态页面

| 页面 | 路径 | 触发条件 |
|------|------|---------|
| 404 | 不存在的路由 | Vue Router catch-all 自动匹配 |
| 403 | `/403` | 无权限访问时手动跳转 |
| 500 | `/500` | request.js 拦截到 5xx 时自动跳转 |

三个页面风格统一：大号状态码 + 说明文字 + 操作按钮

## 网络离线检测

- App.vue 顶部红色横幅：`📡 网络已断开，请检查网络连接`
- 使用 `navigator.onLine` + `online/offline` 事件监听
- 联网自动消失，无需刷新页面

## 分享功能

- 灵感详情页底部「分享」按钮
- 使用 `document.execCommand('copy')`（兼容 http/https）
- 复制成功提示「链接已复制」

## 图片兜底

- 封面图加载失败时显示默认图片（picsum）
- 通过 `@error` + `imgFallback` 处理函数

## 空状态优化

| 页面 | 原文字 | 新文字 |
|------|--------|--------|
| Index.vue | 暂无该分类灵感 | 📭 还没有灵感 |
| Search.vue | 暂无匹配内容 | 🔍 换个关键词试试 |
| Personal.vue | 暂无发布/收藏 | ✍️ / ⭐ 提示 |

## 操作反馈

| 场景 | 反馈 |
|------|------|
| 收藏/点赞成功 | ElMessage.success |
| 网络异常 | ElMessage.error |
| 未登录操作 | ElMessage.warning 并跳转登录页 |
| API 返回错误 | 显示后端返回的 msg 内容 |

## 错误处理

- request.js 全局 axios 拦截器捕捉 HTTP 4xx/5xx
- 401 → 清除 token 跳转登录页
- 5xx → 跳转 `/500` 页面
- BusinessException（HTTP 200 + code 500）通过 `res.code === 200` 判断

## 路由优化

- scrollBehavior 自动滚动到顶部
- 路由守卫 needAdmin / needLogin
- 404 / 403 / 500 独立兜底路由

## 其他

- 回到顶部替换为自定义实现（避免 z-index 冲突）
- 骨架卡片 CSS 复用 `.s-line` 类
- 所有优化不依赖第三方库（仅使用 Element Plus 和 Vue 原生）
