**socket 模式（本地测试）：** 另一个终端先 `nc -lk 9999`

**RocketMQ 模式（对接真实数据）：** 确保 RocketMQ + 后端服务都在跑，启动参数传 `--source rocketmq`：

```bash
cd /Users/lee/Desktop/Project/Study\ Project/Codex/inspire-ai-preview/backend/flink-jobs

/usr/local/bin/mvn spring-boot:run \
  -Dspring-boot.run.jvmArguments="--add-opens java.base/java.lang=ALL-UNNAMED --add-opens java.base/java.util=ALL-UNNAMED" \
  -Dspring-boot.run.arguments="--source rocketmq --host localhost --port 9876"
```

启动后你在浏览器点赞/收藏，Flink 会自动从 `topic_user_behavior` 消费并写入 Redis，不需要手动发数据了。
