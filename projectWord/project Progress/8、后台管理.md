| 接口 | 说明 |
|------|------|
| `POST /api/admin/login` | 管理员登录 |
| `GET /api/admin/dashboard` | 监控大屏 |
| `GET /api/admin/inspire/list` | 全量灵感列表（含搜索、筛选、分页） |
| `PUT /api/admin/inspire/{id}/block` | 下架灵感（deleted=2） |
